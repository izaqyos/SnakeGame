# Snake Game Visual Flow Diagram

## Main Application Flow

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   App Launch        +---->+   MainActivity      +---->+   Authentication    |
|                     |     |   (Login Screen)    |     |   Process           |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------+-----------+
                                     |                            |
                                     v                            v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Registration      |     |   GameActivity      |
                      |   Process           |     |   (Main Game)       |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
                                                      |           |
                                                      v           v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Settings Screen   |<----+   Game Loop         |
                      |                     |     |   Process           |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
                                                            |
                                                            v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   High Scores       |<----+   Game Over         |
                      |   Screen            |     |   Process           |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
```

## Authentication Flow

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   User Input        +---->+   Check Local       +---->+   Valid?            |
|   Credentials       |     |   Storage           |     |   YES: Start Game   |
|                     |     |                     |     |   NO: Check Firebase|
+---------------------+     +---------------------+     +---------------------+
                                                            |
                                                            v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Firebase Check    +---->+   Valid?            |
                      |                     |     |   YES: Save Locally  |
                      |                     |     |   NO: Show Error     |
                      +---------------------+     +---------------------+
```

## Game Loop Flow

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   Game Loop Start   +---->+   Update Game       +---->+   Process Input     |
|                     |     |   State             |     |   (Direction)       |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------------------+
        ^                                                         |
        |                                                         v
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   Draw Frame        |<----+   Update Snake      |<----+   Check Collisions  |
|   On Screen         |     |   Position          |     |                     |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------------------+
                                    ^                             |
                                    |                             v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Place New Food    |<----+   Check Food        |
                      |   If Eaten          |     |   Collision         |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
```

## Game Over Flow

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   Collision         +---->+   Game Over         +---->+   Save High Score   |
|   Detected          |     |   State             |     |   to Firebase       |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------------------+
                                                            |
                                                            v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Show Game Over    +---->+   User Choice       |
                      |   UI                |     |   Restart or Exit   |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
```

## Settings Flow

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   Settings Button   +---->+   Settings Screen   +---->+   Color Selection   |
|   Pressed           |     |                     |     |                     |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------------------+
                                                            |
                                                            v
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Save              +---->+   Return to Game    |
                      |   Preferences       |     |   with New Settings |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
```

## Navigation Flow

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|   Back Button       +---->+   Show Confirmation +---->+   User Choice       |
|   Pressed           |     |   Dialog            |     |   Exit or Continue  |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------------------+
                                                            |
                      +---------------------+     +---------------------+
                      |                     |     |                     |
                      |   Return to         |<----+   Resume Game       |
                      |   Main Menu         |     |                     |
                      |                     |     |                     |
                      +---------------------+     +---------------------+
```
